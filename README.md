# Lab-7-hamburger-project

This project demonstrates **Object-Oriented Programming (OOP) in JavaScript** by creating a constructor function to generate multiple customizable hamburgers.

## Features

- **Constructor function** `Hamburger` to create new hamburgers
- **Seven customizable options**:
  - `bunType` – type of bun (e.g., sesame, brioche)
  - `vegetables` – array of vegetables (e.g., lettuce, tomato, onion)
  - `cheese` – type of cheese (e.g., cheddar, swiss)
  - `sauce` – object with type and optional spicy property
  - `pattyType` – type of patty (beef, chicken, veggie)
  - `pattyCount` – single, double, or triple
  - `extras` – additional toppings (pickles, jalapeños, olives)
- **Method** `.describe()` to output a full description of each hamburger
- **Multiple instances** using the `new` keyword
- **Display on webpage** using DOM manipulation
